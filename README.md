# Contributory-Broadcast-Encryption-with-Efficient-Encryption-and-Short-Cipher-texts
### Swings - Application
### Pre-requisites
- Mysql 5.1
- JDK 8
- Apache Tomacat 8
- Netbeans 8.2
- Mysql Query Browser or  SQLYog
### Steps to Run

- Load SQL into Database
- Open code in Netbeans 8.2
- Build and Deploy code in Apache

### Loading SQL
Using Command Line
```sh
cd Database
mysql -uroot -proot
source <path to sql>
```
